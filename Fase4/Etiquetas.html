Tema 1

En un principio el lenguaje de programación ha sido escencial para realizar diferentes tareas mediiante su automatización, en este caso html buscó crear un lenguaje especializado en la reestructuración de la información, es decir el orden de los elementos de una pagina web, donde normalmente se encuentra la cabecera, barra de navegación, información principal, barra lateral y un pie de pagina, antes se utilizaba la etiqueta table, la cual contenia varias secciones para las diferente partes del sitio web, con lo cual se decidio reinventar las etiquetas como lo es el caso de la etiqueta div, la cual fracciona los componentes de la misma permitiendo visualizar los diversos contenidos destinados para cada una de las partes. no obstante en el codigo de programación la etiqueta div solo servia para separar el contenido más no diferenciarlo uno del otro, por lo tanto html rediseño su codigo complementandolo con las etiquetas que hoy en dia conocemos como lo es la etiqueta header, nav, section, article, aside, entre otros

Es Decir!!

Las etiquetas semánticas son aquellos caracteres o comandos especiales los cuales sirven para determinar una función en el lenguage de programación, a ellas se les atribuye los conceptos primarios con los que se va a trabajar el código, en este caso el de un sitio web con la finalidad de que la interfaz sea más amigable con el usuario, facilitando y optimizando las tareas que se requieran para la construcción del mismo.
